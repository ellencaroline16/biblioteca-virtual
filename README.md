# Biblioteca Virutal:  
Projeto desenvolvido para a disciplina de Métodos de Pesquisa e Ordenação em Estruturas de Dados.

## Sobre o projeto:
Sistema de gerenciamento de uma biblioteca virtual implementado em Java, utilizando diferentes estruturas de dados ao longo das semanas.

## Estruturas utilizadas:
- **LinkedList** — através do gerenciamento do acervo de livros
- **Queue** — fila de espera para livros emprestados
- **Stack** — histórico de navegação do usuário
- **HashMap** — grafo de recomendações de livros

## Estrutura do projeto:
```
src/
├── Livro.java
├── Biblioteca.java
├── GerenciadorFilas.java
├── HistoricoNavegacao.java
├── GrafoRecomendacoes.java
└── Main.java
```
